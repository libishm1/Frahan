Libish (the project owner) elected to honor the GPL on 2026-05-22 in

## Scope of this notice

GPL-3.0 applies to:
- `Frahan.StonePack.GH/Kintsugi/` directory and its descendants.
- Any `Frahan.Kintsugi.Port` C# files that translate functions /
  classes / weights / training-protocol details from the
  PuzzleFusion++ Python source.

GPL-3.0 does NOT apply to:
- The rest of the `Frahan.StonePack.*` codebase, which retains its
  original (non-GPL) licensing.
- Frahan-original algorithms in `Frahan.StonePack.Core` or
  `Frahan.EdgeMatching.Core` that pre-date this addition.
- The geometric `KintsugiAssemblyComponent` 2026-05-22 clean-room
  variant (kept as a fallback path; it never linked GPL code).

## Practical implication

If the full port ships, the resulting `Frahan.StonePack.gha` becomes
GPL-3.0 in distribution because it links the ported port code. Users
of the `.gha` may request the source. Frahan must make the source
available.

The non-GPL parts of Frahan can still be developed independently,
but binary distribution of the COMBINED `.gha` is GPL-3.0 once the
port lands.
